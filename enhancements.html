Animation

CSS Animation is a part of element can change and move with a certain move and duration.
We can do many types of animation and imply it on many group or elements.
Firstly, We style the elements with Animation property and it sub-properties
e.g sub-properties for animation are :

  • animation-name

    Make a certain name for the @keyframes rule

  • animation-duration

    Settings the length of time of an animation in one cycle or one style

  • animation-direction

    Set up the direction of an animation whether it animated forwards, backwards or alternate cycles

  • animation-delay

    Decide how long a delay to start an animation

  • animation-timing-function

    Configures a speed curve of an animation

  • animation-iteration-count

    Set the number of times the animation should run

  • animation-fill-mode

    Settings the style for an element when the animation not moving or playing

After set up all the animations properties, we need to use @keyframes rule.
Keyframes describes how the animation should animated at specific times.
For instance, the very first animation is 0% and 100% is the end.
We can also divide it into 5 like 0%, 25%, 50%, 75% and 100%.
We set the @keyframes up to ourself.

Media Queries

Media queries give us an opportunity to edit the website at a certain size of device.
Since nowadays, there are many type of devices with different sizes (e.g phone, tablet, desktop, laptop)
Media queries help us to build a flexible layout.

For Media Query we use @media syntax, followed by the resolution of the device (e.g min-width, max-width)
