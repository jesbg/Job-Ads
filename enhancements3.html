Log In

Scroll to top button is necessary because it helps visitor on webpage to come back to top after scrolling deep down, specially when the pages is long, therefore i applied this icon on html to recognize it. I also style it on css like change the color, get the transition



and make sure to locate it on the bottom right.
After that we move to java script file, we use get the object element by using querySelector to get the icon



We create function refreshButtonVisibility so when the user scrolls down 150px from the top of the button,
the button will show while when it less than 150px, the button disappear.
Next, we followed just like the example.

We called back the const where we call the icon (scroll to top), we use
addEventListener ("click", () => { }
and write the code to scroll to top (0)
document.body.scrollTop = 0; is for Safari.
document.documentElement.scrollTop = 0; for Chrome, Firefox, IE and Opera.

Next we write document.addEventListener("scroll", () => {
refreshButtonVisibility();
});
so when we scroll, the function refreshButtonVisibility() is working.

I applied this to all pages including Index, Jobs,
Apply, About, and this Enhancements to make visitor easier to scroll back to top.
